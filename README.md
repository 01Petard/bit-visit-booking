# 北京理工大学访客预约系统

一个简洁、美观的访客预约系统单页面应用，支持一键预约、查询记录等功能。

## 🚀 快速开始

### 在线访问

🌐 **直接访问**：[https://bit.bugstack.icu/](https://bit.bugstack.icu/)

### 本地部署

1. **直接打开**

   - 下载 `index.html` 文件
   - 用浏览器直接打开即可使用

2. **本地服务**

   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 或使用 Node.js
   npx http-server
   ```

   然后访问 `http://localhost:8000`

## 📋 功能特性

### 核心功能
- ✅ **访客预约**：支持选择校区、填写访客信息、选择预约时间（当前时间往后7天）
- ✅ **一键登录**：手机号验证码登录，自动缓存 Token
- ✅ **预约记录查询**：分页查询预约记录，每页显示 4 条
- ✅ **自动重试机制**：Token 过期自动重新登录并继续预约流程
- ✅ **登录状态管理**：实时显示登录状态，支持退出登录

### 用户体验
- 🎨 **现代化 UI 设计**：渐变背景、卡片式布局、流畅动画
- 📱 **响应式设计**：完美适配桌面端、平板和移动设备
- 🔔 **智能提示**：右上角抽屉式消息提示，3秒自动消失
- ✨ **交互动画**：按钮悬停效果、输入框聚焦动画、页面加载动画
- 🔗 **快速链接**：一键跳转官方预约列表和预约详情页面

## 📖 使用说明

### 预约流程

1. **填写预约信息**
   - 选择校区（中关村校区/良乡校区）
   - 填写姓名、身份证号、手机号
   - 选择开始时间和结束时间（必须在当前时间往后7天内）

2. **提交预约**
   - 点击"提交预约"按钮
   - 如果未登录，系统会自动弹出登录框
   - 输入手机号和验证码完成登录
   - 登录成功后自动继续预约流程

3. **查看预约记录**
   - 点击"查询预约记录"按钮
   - 查看分页显示的预约记录
   - 点击"查看预约详情"可跳转到官方详情页

### 登录功能

- **登录**：点击右上角"登录"按钮，输入手机号和验证码
- **退出登录**：点击右上角"退出登录"按钮，清除本地 Token
- **自动登录**：Token 过期时系统会自动提示重新登录

## 📡 API 接口

系统对接以下 API 接口：

1. **获取验证码**：`POST /api/guestuser/getCode`
2. **手机号登录**：`POST /api/guestuser/phoneLogin`
3. **提交预约**：`POST /api/guestuser/submit`
4. **查询预约记录**：`GET /api/guestuser/getBookingRecords`

所有 API 请求会自动携带 Token，Token 过期时会自动重新登录。

## 📱 响应式设计

- **桌面端**（> 768px）：两列布局，充分利用空间
- **平板端**（≤ 768px）：单列布局，优化触摸体验
- **移动端**（≤ 480px）：紧凑布局，字体和间距自适应

## 🔒 数据存储

- **Token 存储**：使用 LocalStorage 存储登录 Token
- **预约参数缓存**：临时缓存预约表单数据，便于重试
- **仅限本域名**：不会影响其他网站的缓存数据
